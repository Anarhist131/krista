# Krista Messenger v0.20

Проект Krista — мессенджер с чатами, каналами, лентой новостей и встроенным музыкальным плеером.

**Код предоставлен исключительно для ознакомления.**  
ПО является опенсорсным, модификации разрешаются, но не гарантируют полную работоспособность, по сравнению с оригиналом.

## Основные возможности
- Гостевой режим без регистрации
- Чаты и каналы с подписками
- Каталог публичных чатов и каналов
- Поиск по никнеймам
- Лента сообщений из каналов
- Музыкальный плеер
- Тёмная/светлая тема, кастомизация размера окна
- WebSocket‑уведомления в реальном времени

## Технологии
- Node.js, Express, MongoDB (GridFS для музыки)
- WebSocket, JWT
- PWA (manifest, service worker)
